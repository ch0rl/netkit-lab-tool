# netkit-lab-tool
A GUI for generating Netkit labs.

## Install
1. `virtualenv venv`
2. `source venv/bin/activate`
3. `pip install -r requirements.txt`

## Running
`python netkit-gui/mainwindow.py <path to json file>`
